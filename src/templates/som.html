Self-organizing maps(SOMs) are unsupervised learning models related to the artificial neu-ral networks literature, known to be powerful data mining tools for visualizing high-dimensional data. A self-organizing map is a type of artificial neural network that is trained using unsupervised learning to provide a low-dimensional representation of a high-dimensional input space, called a map. The main characteristic of SOMs is that the mapping between the input and the output space (map) preserves the topology of the input space

Considering all protein conformations, a distribution vector can be computed for each protein, which stores information about the SA elements’ distribution in the protein’s conformations. This frequency vector is constructed by computing the probability of occurrence of letter the conformational transitions.

Based on the 25 values (probability per letter overall in a protein's conformations), sets of 7 or 58 proteins are analysed using SOM.
