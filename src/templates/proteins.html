help Having 10000 successive conformations represented by RSA vectors we group them in classes by using a hyperparameter named step(s) representing the cardinality of each class. The step induces a new hyperparameter: the number of clusters to be used for the clustering process (K). For example, a step = 2500 would mean that we are expecting K=4 classes, namely:
  • Class 1- conformations from 1 to 2500;
  • Class 2- conformations from 2501 to 5000;
  • Class 3- conformations from 5001 to 7500;
  • Class 4- conformations from 7501 to 10000.
